# GPU-Z Log Visualizer

GPU-Zのログファイル（CSV/TXT形式）をブラウザ上で解析し、インタラクティブな折れ線グラフとして可視化するWebアプリケーションです。すべてブラウザ上で動作し、バックエンドサーバーは不要です。

## 特徴
- **Shift-JIS対応**: 日本語環境のGPU-Zログも文字化けせずに読み込めます。
- **ドラッグ＆ドロップ**: 画面のどこにでもファイルをドロップするだけで解析が始まります。
- **ヘッダー分割**: ログ記録の一時停止・再開によって複数回ヘッダーが出現するファイルも、適切に独立したデータセットとして分割します。
- **マルチY軸グラフ**: 各カラムが独立したY軸を持ち、異なる単位のデータ（クロック数や温度など）を重ねて表示できます。
- **インタラクティブな操作**: マウスホイールによるズームや、Ctrl+ドラッグによるパン操作が可能です。Undo/Redoやオートフィット機能も備えています。
- **リアルタイム統計情報**: グラフの表示範囲（X軸）に連動して、最大・最小・平均・中央値をリアルタイムに計算し表示します。

## ビルド方法

開発環境には Node.js が必要です。

```bash
# 依存パッケージのインストール
npm install

# ビルドの実行（Viteによる単一HTMLファイルの生成）
npx vite build
```

ビルドが完了すると、`dist/index.html` が生成されます。このファイルはすべてのCSSやJavaScriptをインラインで含んでおり、ダブルクリックしてブラウザで開くだけで動作します。

## 開発サーバーの起動

開発中にリアルタイムで変更を確認する場合は、以下のコマンドを実行します。

```bash
npx vite dev
```

## テストの実行

Vitestを使用した単体テストを実行するには、以下のコマンドを使用します。

```bash
npx vitest run
```

## 使用方法

1. アプリケーションを開きます（`dist/index.html`をブラウザにドロップするか、開発サーバーにアクセスします）。
2. 解析したいGPU-Zのログファイル（`.csv` または `.txt`）を画面内にドラッグ＆ドロップします。複数ファイルの同時読み込みも可能です。
3. **左サイドバー（上部）**: 読み込まれたデータセット（ファイル名 > タイムスタンプ）のチェックボックスをオンにして有効化します。
4. **左サイドバー（下部）**: グラフに表示したいカラム（温度、クロックなど）のチェックボックスをオンにします。
5. **メインエリア**: 選択したデータセットとカラムに基づく折れ線グラフが表示されます。
   - `マウスホイール`: カーソル位置を中心にX軸をズーム
   - `Shift + マウスホイール`: カーソル位置を中心にY軸をズーム
   - `Ctrl + マウスドラッグ`: X軸に沿ってパン（移動）
   - 各グラフ上部のツールバーから、ズームのUndo/Redo、表示のリセット、ローカルまたは全体に対するオートフィットが行えます。
6. **右サイドバー**: グラフの現在の表示範囲に基づいた統計情報（最大、最小、平均、中央値）が表形式で表示されます。グラフをズーム・パンするとリアルタイムに更新されます。
7. **下部パネル**: アプリケーションの実行ログ（ファイルの読み込み成功、パース状況、エラーなど）が表示されます。
